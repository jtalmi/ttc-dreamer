# TTC Dreamer

Working title for the Toronto Transit Sandbox: a desktop-first web app for
Toronto transit fans to create, edit, and share custom TTC rapid transit
proposals on top of a preloaded Toronto map.

## Status

The repo is bootstrapped, but GSD project initialization has not been run yet.
The source product docs live under `docs/product/`, and the app itself is still
at the neutral scaffold stage.

## Stack

- Next.js 16 App Router
- React 19
- TypeScript 5
- Tailwind CSS 4
- GSD planning workflow via `.claude/get-shit-done`

## Commands

Install dependencies:

```bash
npm install
```

Run the app:

```bash
npm run dev
```

Validate the scaffold:

```bash
npm run lint
npm run typecheck
npm run build
```

## Workflow

1. Refine product direction in `docs/product/*` if needed.
2. Run GSD project initialization from `docs/product/gsd-idea.md`.
3. Use generated `.planning/` docs as the source of truth for phase work.
4. Start with Phase 1 only: app shell, map canvas, toolbar scaffold, collapsible sidebar scaffold, proposal state model, and baseline toggle shell.
