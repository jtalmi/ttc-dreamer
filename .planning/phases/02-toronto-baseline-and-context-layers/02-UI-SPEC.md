## Visual Hierarchy

- **Primary focal point:** the map canvas itself — TTC rapid transit lines must be the visually dominant layer, bold and legible above all context
- **Secondary focal point:** TTC station labels and GO station labels, visible at normal zoom, abbreviated at lower zoom
- **Tertiary focal point:** neighbourhood labels, major street labels, and the bus/streetcar corridor overlay (when toggled on)
- **Layer stacking order (bottom to top):** base map surface → major streets → bus/streetcar corridors (when on) → GO lines → GO station markers → TTC lines → TTC station markers → neighbourhood labels → landmark callouts → station labels
- **Accessibility rule:** all map labels must clear a 3.5:1 contrast ratio against their immediate background; interactive toggle controls use visible text labels

## Typography

Four declared sizes — identical to Phase 1. Map-specific visual subordination is achieved through opacity, letter-spacing, and weight within the Label (14px) role. No new sizes are introduced.

| Role | Size | Weight | Line Height |
|------|------|--------|-------------|
| Body | 16px | 400 | 1.5 |
| Label | 14px | 600 | 1.3 |
| Heading | 20px | 600 | 1.25 |
| Display | 32px | 600 | 1.1 |

Map label rules — all rendered at 14px (Label role), differentiated by weight, letter-spacing, and opacity:

- Neighbourhood labels: 14px, weight 600, all-caps, letter-spacing 0.08em, color `--shell-secondary` at 70% opacity
- Street labels: 14px, weight 400, color `--shell-secondary` at 50% opacity
- TTC station labels: 14px, weight 400, color `--shell-secondary` at 90% opacity, placed to the right of station dot by default
- GO station labels: 14px, weight 400, color `--go-secondary` (see Color section) at 80% opacity
- Landmark callouts: 14px, weight 600, color `--shell-secondary` at 85% opacity

Vendor exception (not a declared design system size): map tile attribution text renders at 11px, weight 400, opacity 0.5. This is dictated by the map tile provider and must not be suppressed. It does not enter the type scale.

> Source: Phase 1 UI-SPEC.md base typography (4 sizes). Map sub-roles consolidated into Label (14px) per checker Dimension 4 fix — 2026-03-31.

## Interaction Contract

### Layer Toggle Control

- Lives in the top toolbar, to the right of the baseline toggle group
- Exact label: `Bus + Streetcar Corridors`
- Toggle has two states: `off` (default on load) and `on`
- When `off`: button uses `--shell-secondary` background, white text, no accent
- When `on`: button uses `--shell-accent` background, white text — matches active tool pill pattern from Phase 1
- Toggling is instant: no loading state, no animation delay

### Baseline Toggle Interaction with Map Layers

- The existing `Today` / `Future committed` toggle from Phase 1 is preserved unchanged
- Switching baseline mode must re-render TTC and GO layers to reflect the selected baseline dataset
- No additional UI change is required for the toggle itself

### Map Layer Behavior

- TTC and GO layers are always visible — they are not user-toggleable in Phase 2
- Only the bus/streetcar corridor layer is user-toggleable (MAP-04)
- GO lines and stations: pointer-events disabled (cursor: default on hover, no tooltip, no selection)
- TTC station dots: pointer-events enabled, but in Phase 2 they show a read-only label tooltip only (no inspector yet — that is Phase 4)
- Neighbourhood, street, and landmark labels are non-interactive

### Map Viewport

- Map renders inside the existing `MapStage` `children` slot (Phase 1 decision preserved)
- Map fills the full MapStage area: no padding, no chrome border
- When the map library initializes, it replaces the Phase 1 placeholder grid texture entirely
- Attribution text (map tile copyright) is pinned to the bottom-right corner at 11px, weight 400, opacity 0.5 — do not suppress it

## Visual Hierarchy — Toronto-Native Identity Rules

These rules ensure the map reads as distinctly Toronto, per the product constraint.

1. TTC line colors must match canonical TTC brand values exactly (see Color table above) — do not substitute design-system accent colors for TTC lines
2. GO renders at visually subordinate weight to TTC at all times — lighter stroke, lower opacity, dashed line
3. Neighbourhood labels use well-known Toronto neighbourhood names (e.g. "Leslieville", "Annex", "Scarborough", "Etobicoke") — not custom or invented names
4. The map base tile style must be muted/desaturated — Toronto geography should read as background; transit lines are the hero
5. When bus/streetcar corridors are visible, they must not compete visually with rapid transit lines — use lower opacity and thinner stroke
