  <behavior>
    **Stat functions (TDD targets):**
    - computeLineLength: 3-waypoint subway line across Toronto returns ~12.3 km; line with <2 waypoints returns 0
    - computeTravelTime: 12.3 km subway at 60 km/h = ~12 min; 0 km = 0 min
    - computeAvgStopSpacing: line with 3 stations and 12.3 km = ~6.2 km; line with <2 stations returns null
    - computeLineCost: 12.3 km subway at $250M/km = ~3075 M
    - computeLineRidership: 3 stations subway at 2000/station = 6000, rounded to nearest 100 = 6000
    - computeProposalStats: aggregates across multiple lines; empty proposal returns zeroes
    - resolveNeighbourhood: finds nearest neighbourhood within 3km; falls back to coordinates when beyond 3km

    **Reducer actions:**
    - inspectElement({id, elementType: "line"}) sets inspectedElementId=id and sidebarPanel="inspect-line"
    - inspectElement({id, elementType: "station"}) sets inspectedElementId=id and sidebarPanel="inspect-station"
    - closeInspector clears inspectedElementId and sets sidebarPanel="list"
    - toggleComparisonMode flips comparisonMode boolean
    - None of these three actions appear in HISTORY_ACTIONS (chrome-only, not draft mutations)
  </behavior>
