       - `exportMapAsPng(mapRef: React.RefObject<MapRef | null>, filename: string): Promise<void>` -- calls `mapRef.current?.getCanvas().toDataURL("image/png")`, creates an anchor element with download attribute, clicks it. Throws if canvas is null.
