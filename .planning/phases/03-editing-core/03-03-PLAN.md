**1. Add delete and naming/coloring actions to proposal-state.ts:**

- `deleteLine`: payload `{ lineId: string }` — removes the line from `draft.lines`. Also removes all stations that only belong to this line (`lineIds.length === 1 && lineIds[0] === lineId`). For stations shared with other lines, remove this lineId from their `lineIds` array. Clear `selectedElementId` if it matched the deleted line. In `HISTORY_ACTIONS`.

- `deleteStation`: payload `{ stationId: string }` — removes the station from `draft.stations`. Removes the stationId from all lines' `stationIds` arrays. Clear `selectedElementId` if it matched. In `HISTORY_ACTIONS`.

- `deleteSelected`: no payload — reads `chrome.selectedElementId`. Determines if it's a line or station (check `draft.lines` first, then `draft.stations`). If it's a baseline TTC element (selectedElementId is not found in proposal lines or stations), do nothing silently. If found: does NOT directly delete — instead sets a pending deletion state in chrome. Add `pendingDeletion: { type: "line" | "station"; id: string; name: string; isShared?: boolean; sharedLineCount?: number } | null` to `EditorChromeState`. This triggers the confirmation dialog.

- `confirmDeletion`: no payload — reads `chrome.pendingDeletion`, dispatches the actual `deleteLine` or `deleteStation`, clears `pendingDeletion`. In `HISTORY_ACTIONS`.

- `cancelDeletion`: no payload — clears `pendingDeletion`.

- `updateLineName`: payload `{ lineId: string; name: string }` — updates the line's name. In `HISTORY_ACTIONS`.

- `updateLineColor`: payload `{ lineId: string; color: string }` — updates the line's color. In `HISTORY_ACTIONS`.

- `clearProposal`: no payload — resets `draft.lines` and `draft.stations` to empty arrays. Clears all chrome editing state. In `HISTORY_ACTIONS`. Also triggered via confirmation dialog.

**4. Update editor-shell.tsx** — wire keyboard shortcuts and confirmation dialog:

- Add `useEffect` for keyboard undo/redo/delete handler per RESEARCH code example:
  ```
  useEffect(() => {
    const handleKeyDown = (e: KeyboardEvent) => {
      const isMac = navigator.platform.startsWith("Mac");
      const mod = isMac ? e.metaKey : e.ctrlKey;
      if (mod && e.key === "z" && !e.shiftKey) {
        e.preventDefault();
        dispatch({ type: "undo" });
      }
      if (mod && e.key === "z" && e.shiftKey) {
        e.preventDefault();
        dispatch({ type: "redo" });
      }
      if ((e.key === "Backspace" || e.key === "Delete") && chrome.selectedElementId) {
        e.preventDefault();
        dispatch({ type: "deleteSelected" });
      }
      if (e.key === "Escape" && chrome.drawingSession) {
        dispatch({ type: "cancelDrawing" });
      }
    };
    window.addEventListener("keydown", handleKeyDown);
    return () => window.removeEventListener("keydown", handleKeyDown);
  }, [dispatch, chrome.selectedElementId, chrome.drawingSession]);
  ```
