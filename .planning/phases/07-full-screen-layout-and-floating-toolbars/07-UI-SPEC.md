## Layout Contract

### Full-Screen Shell

Remove `TopToolbar` from `EditorFrame`. The outer container (`100vw × 100vh, overflow: hidden`) becomes a pure stacking context with `position: relative`.

```
┌─────────────────────────────────────────────────┐
│  MapStage (position: absolute, inset: 0)        │
│  ┌──────────┐                                   │
│  │ Drawing  │                    ┌────────────┐ │
│  │ Toolbar  │                    │   Layer    │ │
│  │ (left,   │                    │  Picker    │ │
│  │ center)  │                    │ (btm-right)│ │
│  └──────────┘                    └────────────┘ │
│                                      [›] chevron │
└─────────────────────────────────────────────────┘
                                    ┌──────────────┐
                                    │   Sidebar    │
                                    │   (320px,    │
                                    │ position:    │
                                    │  absolute    │
                                    │  right: 0)   │
                                    └──────────────┘
```

### Floating Drawing Toolbar

- **Position:** `position: absolute; left: var(--space-lg); top: 50%; transform: translateY(-50%); z-index: var(--z-floating-toolbar)`
- **Shape:** Pill — `border-radius: 12px`
- **Background:** `var(--floating-toolbar-bg)` + `var(--floating-toolbar-shadow)`
- **Layout:** `flex-direction: column; gap: var(--space-xs); padding: var(--space-sm)`
- **pointer-events:** `pointer-events: none` on the wrapper; `pointer-events: auto` on each button
- **Buttons:** 3 icon-only buttons (Select, Draw, Add Line), each 44px × 44px
- **Active state:** `background-color: var(--shell-accent); border-radius: 8px`
- **Tooltip:** Plain `title` attribute (native browser tooltip). No custom tooltip in Phase 7.
- **Divider:** 1px `rgba(243, 238, 229, 0.15)` horizontal rule between Add Line and the Select/Draw group (Add Line is semantically separate — it mutates proposal state rather than switching edit mode)

### Floating Layer Picker

- **Position:** `position: absolute; bottom: var(--space-lg); right: calc(var(--space-lg) + 8px); z-index: var(--z-floating-toolbar)`
- **Shape:** Card — `border-radius: 8px`
- **Background:** `var(--layer-picker-bg)` + `var(--layer-picker-shadow)`
- **Layout:** `flex-direction: column; gap: var(--space-xs); padding: var(--space-md)`
- **pointer-events:** `pointer-events: none` on wrapper; `pointer-events: auto` on each row
- **Contents:** 3 toggle rows — Baseline (Today/Future), Corridors (on/off), Comparison (on/off, disabled when no lines)
- **Toggle row layout:** 32px icon + label (14px/400) + toggle switch on right, `align-items: center; gap: var(--space-sm); height: 36px`
- **Toggle control:** Plain `<button aria-pressed>` styled as pill toggle; active = `var(--shell-accent)` bg; inactive = `rgba(243, 238, 229, 0.15)` bg

### Sidebar Panel

- **Position:** `position: absolute; top: 0; right: 0; bottom: 0; z-index: var(--z-sidebar)`
- **Width:** 320px (unchanged)
- **Background:** `var(--shell-secondary)`
- **Transition:** `transform 0.2s ease` — open: `translateX(0)`, closed: `translateX(320px)`
- **Sidebar toggle button:**
  - Position: `position: absolute; top: 50%; right: 0; transform: translateY(-50%) translateX(100%); z-index: var(--z-sidebar-toggle)` when sidebar is closed (chevron emerges from right edge of map)
  - When sidebar is open: button is inside the sidebar left edge (or overlapping)
  - Size: 32px × 32px, `border-radius: 6px 0 0 6px` (left half-pill)
  - Background: `var(--shell-secondary)`
  - Icon: `ChevronRight` (closed) / `ChevronLeft` (open) from lucide-react, 16px

### Sidebar Header (replaces TopToolbar title/share area)

- **Height:** 52px, `flex-shrink: 0`
- **Background:** `rgba(243, 238, 229, 0.06)` (subtle separation from content)
- **Border-bottom:** `1px solid var(--inspector-divider)`
- **Layout:** `flex; align-items: center; padding: 0 var(--space-md); gap: var(--space-sm)`
- **Proposal title:** Inline editable span → input (same pattern as existing TopToolbar). 20px/600, click to edit. Max-width 180px with ellipsis.
- **Share button:** Right-aligned, `padding: var(--space-xs) var(--space-md)`, accent bg, 14px/600
